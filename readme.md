# Digital Wallet System (Python CLI)

A simple, educational command-line based digital wallet application that allows two users to manage virtual money with deposits, withdrawals, and peer-to-peer transfers.

Perfect for learning Object-Oriented Programming (OOP), transaction logging, and building interactive Python applications.

## Features

- Create two named wallets
- Deposit & withdraw money
- Transfer funds between wallets
- Real-time balance tracking
- Full transaction history with:
  - Timestamp
  - Action type (Deposit / Withdrawal / Send / Receive)
  - Amount
  - Notes (e.g., "Sent to Bob")
  - Unique 8-character Transaction ID (TX-ID)
- User-friendly menu interface
